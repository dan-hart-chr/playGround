# Dimension
## Assignment Description
Create an HTML page that exactly replicates the design of the marketing banner shown above. Use separate HTML and CSS files.

### 1. Banner Container
- **Dimensions**: Fixed width of `300px` and height of `600px`.
- **Background**: Hex color `#c6c0af`.
- **Layout**: The container should center its content.

### 2. Main Image
- **Size**: `230px` width by `250px` height.
- **Positioning**:
  - Horizontally centered.
  - Placed `50px` from the top edge of the banner.
- **Source**: Use a placeholder image or a relevant coffee asset.

### 3. Primary Text
- **Content**: "The Coffee Shop"
- **Typography**: Font family `Verdana`, size `1.3rem`, weight `regular` (normal).
- **Positioning**: Starts `50px` directly below the main image.

### 4. Footer Text
- **Content**: "Where Taste Matters"
- **Typography**: Font family `Verdana`, size `1.2rem`, weight `bold`.
- **Positioning**: Placed `50px` from the bottom edge of the banner.
